0.1 Introduction

We follow the paper “On ordered semigroups” by N. G. Alimov.

0.2 Content

Definition 1
#

An ordered semigroup \(S\) is a semigroup with a linear order such that for all \(a,b,c\in S\) such that \(a {\lt} b\)

  1. \(a + c {\lt} b + c\)

  2. \(c + a {\lt} c + b\)

Theorem 1
#

An ordered semigroup is cancellative.

Proof ▶

Let \(S\) be an ordered semigroup. Let \(a,b,c\in S\).

Assume that \(a + c = b + c\). Then if \(a {\lt} b\) we have that \(a + c {\lt} b + c\) which is a contradiction. Likewise, if \(b {\lt} a\), we have that \(b + c {\lt} a + c\) which is a contradiction. Therefore, \(a = b\).

Assume that \(c + a = c + b\). Then if \(a {\lt} b\) we have that \(c + a {\lt} c + b\) which is a contradiction. Likewise, if \(b {\lt} a\), we have that \(c + b {\lt} c + a\) which is a contradiction. Therefore, \(a = b\).

Definition 2
#

An element \(a\) of an ordered semigroup \(S\) is positive if for all \(x\in S\), \(a+x {\gt} x\).

Definition 3
#

An element \(a\) of an ordered semigroup \(S\) is negative if for all \(x\in S\), \(a+x {\lt} x\).

Definition 4
#

An element \(a\) of an ordered semigroup \(S\) is zero if for all \(x\in S\), \(a+x = x\).

Theorem 2
#

Each element \(a\) of an ordered semigroup \(S\) is either positive, negative, or zero.

Proof ▶

Let \(a\in S\) and \(b\in S\). Since the \(S\) is a linear order we have one of the following cases.

The first case is that \(b + a {\gt} b\). Then for all \(x\in S\) we have that \(b + a + x {\gt} b + x\) and so \(a + x {\gt} x\). Therefore, \(a\) is positive.

The second case is that \(b + a {\lt} b\). Then for all \(x\in S\) we have that \(b + a + x {\lt} b + x\) and so \(a + x {\lt} x\). Therefore, \(a\) is negative.

The last case is that \(b + a = b\). Then for all \(x\in S\) we have that \(b + a + x = b + x\) and so \(a + x = x\). Therefore, \(a\) is zero.

Corollary 1
#

Let \(a\) be an element of an ordered semigroup \(S\).

If there exists a \(b\in S\) such that \(b + a {\gt} b\), \(a\) is positive.

If there exists a \(b\in S\) such that \(b + a {\lt} b\), \(a\) is negative.

If there exists a \(b\in S\) such that \(b + a = b\), \(a\) is zero.

Theorem 3
#

Let \(a\) and \(b\) be elements of an ordered semigroup \(S\).

If \(a\) is negative and \(b\) is positive, then \(a {\lt} b\).

Proof ▶

Let \(a\) be negative and \(b\) be positive. Then for all \(x\in S\) we have that \(a + x {\lt} x\). Likewise, for all \(x\in S\) we have that \(b + x {\gt} x\). Therefore, \(a + x {\lt} x {\lt} b + x\) and so \(a {\lt} b\).

We endow our ordered semigroup with scalar multiplication by the positive natural numbers.

Lemma 1
#

Let \(a\) and \(b\) be elements of an ordered semigroup \(S\).

For all \(n {\gt} 1\), \(n(a+b) = a + (n-1)(b+a) + b\).

Proof ▶

Let \(n=2\). Then \(2(a+b) = a + (2-1)(b+a) + b\).

Assume that the statement holds for \(n\). Then we have that \((n+1)(a+b) = a + b + n(a+b) = a + b + a + (n-1)(b+a) + b = a + n(b+a) + b\).

Lemma 2
#

Let \(a\) and \(b\) be elements of an ordered semigroup \(S\).

If \(a + b {\lt} b + a\), then for any \(n\in \mathbb {N}^+\), we have that

\[ na + nb {\lt} n(a+b) {\lt} n(b+a) {\lt} nb + na \]
Proof ▶

If \(n=1\), then we are done.

Assume that the statement holds for \(n\). Then we have that

\begin{align} (n+1)a + (n+1)b & = a + na + nb + b \\ \text{by the induction hypothesis}\\ & {\lt} a + n(a + b) + b \\ \text{since $a+b {\lt} b+a$}\\ & {\lt} a + n(b + a) + b \\ \text{by the previous lemma}\\ & = (n+1)(a + b) \\ & {\lt} (n+1)(b + a) \\ \text{by the previous lemma}\\ & = b + n(a + b) + a \\ & {\lt} b + n(b + a) + a \\ \text{by the induction hypothesis}\\ & {\lt} b + nb + na + a \\ & = (n+1)b + (n+1)a \end{align}
Definition 5
#

Let \(a\) and \(b\) be elements of an ordered semigroup \(S\) that are not zero.

Then \(a\) is said to be Archimedean with respect to \(b\) if there exists an \(N\in \mathbb {N}^+\) such that for \(n {\gt} N\), the inequality \(b {\lt} na\) holds if \(b\) is positive, and the inequality \(b {\gt} na\) holds if \(b\) is negative.

Definition 6
#

An ordered semigroup is Archimedean if any two of its elements of the same sign are mutually Archimedean.

Definition 7
#

Let \(a\) and \(b\) be elements of an ordered semigroup \(S\).

Then \(a\) and \(b\) are said to form an anomalous pair if for all \(n\in \mathbb {N}^+\), one of the following holds

\begin{align} na {\lt} nb {\lt} (n+1)a \\ na {\gt} nb {\gt} (n+1)a \end{align}
Theorem 4
#

If \(S\) is a non-Archimedean ordered semigroup, then there exists an anomalous pair.

Proof ▶

Since \(S\) is non-Archimedean, there exists \(a,b\in S\) such that \(a\) and \(b\) have the sign and are not mutually Archimedean.

First, we look at the case where \(a\) and \(b\) are positive. Then since they are not mutually Archimedean, without loss of generality, for all \(n\in \mathbb {N}^+\), \(na {\lt} b\).

Then we either have that \(a + b \le b + a\) or that \(b+a\le a + b\). In the first case, we have that

\[ na + nb \le n(a+b) \le nb + na \]

which means that, since \(na {\lt} b\),

\[ nb {\lt} n(a+b) {\lt} (n+1)b \]

And so \(b\) and \(a+b\) form an anomalous pair.

The other cases follow similarly.

Theorem 5
#

An ordered semigroup without anomalous pairs is an Archimedean commutative semigroup.
